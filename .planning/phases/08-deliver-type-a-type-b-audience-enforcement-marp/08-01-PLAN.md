## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User cwd → deliver.cjs | User-controlled `.planning/workstreams/**/*.md` content read by deliver.cjs — not sanitized; treated as own user input |
| deliver.cjs → atomicWriteFileSync output path | output path constructed from `path.join(cwd, '.planning', 'deliverables', 'type-a', '{artifactKey}.md')` — keys are constants from TYPE_A_ARTIFACTS, not user-supplied |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01-01 | Information Disclosure | deliver.cjs SYNTHESIS_MAP source paths | mitigate | Source artifact paths are constants in SYNTHESIS_MAP — no user-supplied path interpolation. Path injection impossible because `src.artifact` is a literal string in the constant table. |
| T-08-01-02 | Tampering | conditionalProse regex on user-controlled body | mitigate | Regex `/<!--\s*BEGIN business_model:\s*(\w+)\s*-->[\s\S]*?<!--\s*END business_model:\s*\1\s*-->/g` uses backreference `\1` to enforce well-formed open/close markers. Mismatched/unbalanced markers cause no match (block left intact, not stripped) — safe failure mode. |
